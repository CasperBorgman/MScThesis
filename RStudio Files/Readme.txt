Scripts created and adapted by Casper Borgman (2019)

BulkClipPlots - clip vegetation plots with LiDAR data to get LiDAR data for a vegetation plot
BulkNormalization- normalize data to remove ground height, and remove ground points.
ExtractAHN - interact with AHN-2 API to download AHN data
MakeShapefile - create a shapefile of the vegetation database with plot locations
